## 핵심 원칙

> **"남의 언어를 잘 쓰는 게 목표가 아니다. 내 언어를 갖는 게 목표다."**

## 2대 하위 영역

1. **감정어 (Emotional Vocabulary)**
   - 공감각 치환: 추상 감정 → 온도/무게/색/냄새/질감
   - 예: "슬프다" → "뜨겁다" / "무겁다" / "투명하다"

2. **문장 형식 (Syntactic Identity)**
   - 통사 선호 패턴 발견 (부사 선호, 전치사 선호 등)
   - 나만의 언어 지문 (Language Fingerprint)

## 통합 구조
