## 시스템 아키텍처

Skillset6 is processed through three stages:

1. **Feed** – 정보 수집
2. **Digest** – 구조화
3. **Expression** – 표현

```
Input → Processing → Output
  ↓         ↓          ↓
Feed → Digest → Expression
```
